Patches

To determine which version of Twister OS you're currently using, check the README file on the Desktop. For version 1.5.2 and newer, simply open a terminal and type twistver

Please patch in order, do not skip patches.

Twister OS Patcher has been included in version 1.6 patch


To download Twister OS Patcher copy the following into terminal.

git clone https://github.com/phoenixbyrd/TwistUP-OS.git && mv /home/pi/TwistUP /home/pi/patcher
