Patches

To determine which version of Twister OS you're currently using simply open a terminal and type twistver

Please patch in order, do not skip patches.

To download Twister OS Patcher copy the following into terminal.

git clone https://github.com/phoenixbyrd/TwistUP-Lite.git && mv /home/pi/TwistUP-Lite /home/pi/patcher
